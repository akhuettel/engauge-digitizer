Adding Guideline Along Constant Coordinate Value

Similar to grid lines, one or more guidelines can be added along constant-coordinate values for alignment.

The following steps add a guideline along a constant-coordinate value:

  • Select Digitize Guideline mode using the Digitize Tools toolbar.
  • Click where the guideline is to be added.
  • Drag in a direction parallel to the axis of the desired coordinate.
  • Once the drag has traveled sufficiently far, a new guideline will appear.

After a guideline has been created, its position can be adjusted while in Guidelines Tool mode. In other modes the guidelines are fixed.
